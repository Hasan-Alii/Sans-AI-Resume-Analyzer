üîó **[Live Demo](https://sans-resume-analyzer-4c1se.puter.site/)**

## üöÄ Features

* **AI-Powered Analysis:** Leverages LLMs via Puter.js to critique resumes and suggest improvements.
* **PDF Parsing:** Native PDF text extraction using `pdfjs-dist`.
* **Drag & Drop Interface:** User-friendly file uploader built with `react-dropzone`.
* **Secure Authentication:** Integrated user login and management via Puter Auth.
* **Responsive UI:** Fully responsive design (mobile/desktop) using Tailwind CSS.
* **Efficient State Management:** Global state handling with Zustand.

## üì¶ Installation

To run this project locally, follow these steps:

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/your-username/ai-resume-analyzer.git](https://github.com/your-username/ai-resume-analyzer.git)
    cd ai-resume-analyzer
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
